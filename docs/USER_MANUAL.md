# Руководство пользователя

Приложение показывает видео с камеры и точку взгляда на виртуальном экране. Используются MediaPipe (лицо/глаза) и свой калибратор (Ridge). Внешние библиотеки трекинга не требуются.

## Установка

```bash
pip install -r requirements.txt
```

## Запуск

```bash
python -m gui.app
```

## Интерфейс

- **Калибровка (файл)** — путь к .pkl. «Загрузить» — загрузить сохранённую калибровку; «Сохранить» — сохранить после калибровки.
- **Камера** — номер устройства (0, 1, …).
- **Старт** — запуск. Если калибровка не загружена, сначала идёт калибровка по точкам (смотрите в зелёный круг). Затем отображается траектория взгляда.
- **Стоп**, **Выход** — остановка и выход.

Рекомендуется после первой калибровки нажать «Сохранить» и в следующий раз загружать файл.

## Неполадки

- Лицо не обнаружено — освещение, лицо в кадре.
- Точка прыгает — перекалибровать и сохранить заново.
