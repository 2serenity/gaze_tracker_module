# Модуль определения направления взгляда на мониторе

Визуализация и API для определения направления взгляда пользователя на экране. **Без внешних библиотек трекинга:** свой экстрактор на MediaPipe Face Mesh и калибратор на Ridge (scikit-learn).

## Требования

- Python 3.10+

## Установка

```bash
cd gaze_tracker_module
pip install -r requirements.txt
```

## Запуск

### Графическое приложение

```bash
python -m gui.app
```

При первом запуске пройдёт калибровка по точкам на экране (смотрите в зелёный круг). Затем можно нажать «Сохранить» и сохранить калибровку в `saved_models/calib.pkl`. При следующем запуске укажите этот файл и нажмите «Загрузить» — фаза калибровки будет пропущена.

### API

После калибровки в GUI и сохранения файла:

```bash
python -m api.main --calib saved_models/calib.pkl --port 8000
```

Без `--calib` API работает, но предсказания без калибровки (центр экрана).

## Состав проекта

- **core/** — экстрактор признаков (MediaPipe), калибратор (Ridge)
- **gui/** — графическое приложение
- **api/** — REST API
- **docs/** — документация
